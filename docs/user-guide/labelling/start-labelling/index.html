To start labeling your data in Visioncloud, navigate to your dataset and open an image you want to annotate. Once the image is open, you will see the labeling interface with various tools and options.

Labels‚Äã

The labels dropdown allows you to select the current label for your annotations. You can choose from the predefined labels you set up in the dataset. Additionally, you can choose to hide or show labels on the image for better visibility.

Label objects‚Äã

This dropdown allows you to select specific label objects within the image. You can choose to show or hide these objects for better visibility. Also, you can delete selected label objects.

Geometry Tools‚Äã

Tools for creating precise spatial annotations:

  • Polygon Tool: Create multi-point shapes to outline objects with irregular boundaries.
  • Circle Tool: Draw circular annotations with adjustable radius.
  • Bounding Box Tool: Create rectangular regions around objects.
  • Oriented Bounding Box: Draw rotated rectangles for better fit to angled objects.
  • Point Tool: Place precise keypoints or landmarks.

Advanced features‚Äã

Specialized capabilities for efficient annotation:

  • Label Presets: Save and reuse common annotation configurations
  • Batch Operations: Apply classifications to multiple annotations simultaneously
