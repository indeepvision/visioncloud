Managing Images

Open an image​

Click on the image Filename to open the image viewer.

You can navigate between images using the arrow buttons or keyboard arrows. The image viewer displays image details, annotations, and provides tools for labeling.

Additionally, you can zoom in/out and pan around the image for detailed inspection.

Image Actions​

If you click on the three dots in the top right corner on the image viewer, you can see the image actions to be performed on the current image.

You can see them also and perform bulk actions in a dataset, when selecting one or more images.

The available actions include:

  • Classify: Assign classification labels to images
  • Unlabel: Clear all annotations from selected images
  • Ignore/Include: Mark images to be excluded from or included in training
  • Delete: Remove images from the dataset
  • Move: Transfer images to another dataset.
  • Download: Download selected images.

Image Selection Tools​

To manage images efficiently, the platform provides several selection tools:

  • Bulk Select: Choose multiple images at once.
  • Filter Options: Find images based on label status, creation date, etc.
  • Sort Options: Order by name, date, size, or other criteria.

Advanced Filtering​

The platform offers powerful filtering capabilities to help you find specific images:

Filter Categories​

  • Label Status: Filter by labeled, unlabeled, or partially labeled images.
  • Label Content: Filter by images containing specific labels.
  • Image status: Filter by images ignored or included in training.
  • Image Properties: Filter by size.

Export Dataset​

You can export images and annotations from a dataset using the Download button. You have two options:

  • Download selection: Download only the selected images.
  • Download Dataset: Click on the three dots on the right superior corner of the dataset page and click Download dataset. This option allows you to export the entire dataset as a ZIP file including annotations in a JSON file, with the same structure as described in the uploading images section.

Move images​

You can move selected images between datasets:

  1. Select the images you want to move.
  2. Click the Move action button.
  3. Choose the destination dataset.
  4. Select transfer options (copy vs. move, transfer annotations).
  5. Confirm the operation.
