Quickstart

Welcome! This guide will help you discover Vision Cloud in less than 5 minutes. You'll learn how to log in and navigate the interface.

Layout of the application​

To begin, go to Vision Cloud and log in using the credentials de ID-Vision team has provided.


Vision Cloud login screen

After logging in, you'll land on a page where you can select the organization you belong to. Each organization contains its own projects, users, and data.


Select your organization to proceed

When you open VisionCloud you will see the following layout. There are 2 main sections in the application:

PROJECTS​

In the Projects section, you'll see all the projects available within your organization. Read more.

What you can do:​

  • Filter projects using the Add Filter button.
  • Search for a specific project
  • Display only projects within a certain date range.
  • Create a new project from scratch.


View and manage your Vision Cloud projects.

SETTINGS​

The Settings section helps you configure your organization. Read more.

It contains the following sections:

Team​

Manage your organization members from this section.

You can:

  • Edit user roles.
  • Update user details.
  • Attach a user to a project.


Manage users

Connections​

Manage and configure connections with OneVision.


Manage OneVision connections

Billing​

Monitor the status of your Vision Cloud license plans.

You’ll be able to:

  • View usage statistics:
    • Projects used vs available
    • Number of models created
    • Number of uploaded images
    • Remaining training hours
  • Check your current plan


Manage license.
