Labelling

Overview​

The app is a powerful interface for viewing, labeling, and interacting with images in your datasets. It provides an intuitive workspace with specialized tools for annotation, classification, and data visualization.

The app is designed to support multiple workflows including:

  • Image viewing and inspection
  • Precise annotation with multiple tools
  • Multi-user collaborative labeling
  • Output visualization and comparison

Interface Layout​

The app interface is organized into several distinct areas, each serving a specific purpose:

Main Canvas​

The central area displays the current image with all annotations and visualizations:

  • Zoomable View: Easily zoom and pan to focus on specific image areas
  • High-Quality Rendering: Maintains image clarity at all zoom levels
  • Interactive Elements: Select, edit, and manipulate annotations directly on the canvas
  • Real-time Updates: See changes from other users immediately

Toolboxes​

The right sidebar contains specialized toolboxes for different functions:

  • Labels Toolbox: Access and manage dataset labels
  • Label Objects Toolbox: View and organize annotations on the current image
  • Outputs Toolbox: View model predictions when available
  • Output Objects Toolbox: Examine specific model-generated detections
  • Report Toolbox: View metrics and feedback for the current image
  • Info Toolbox: See image metadata and properties
  • Tools Toolbox: Access specialized annotation tools

Carousel​

The bottom area shows thumbnails of dataset images:

  • Quick Navigation: Easily move between images in the dataset
  • Status Indicators: See labeling status and user assignments
  • Filtering Options: Filter images by various criteria
  • Collaborative Awareness: See which images other team members are working on

Bottom Toolbar​

The toolbar at the bottom provides controls for labeling operations:

  • Start/Stop Labeling: Toggle between view and edit modes
  • Label Navigation: Move through the labeling sequence
  • Tool Selection: Choose appropriate tools for different annotation types
  • Undo/Redo: Step backward or forward through recent changes

Display Modes​

The interface supports multiple display modes to accommodate different workflows:

View Mode​

A read-only interface for exploring and analyzing dataset images:

  • Browse images and view existing annotations
  • Examine metadata and properties
  • Review model outputs and predictions
  • Export or share visualization results

Label Mode​

The complete annotation interface with all editing tools:

  • Create and modify annotations using specialized tools
  • Apply classifications and tags
  • Work through labeling tasks sequentially
  • Collaborate with other team members in real-time

Compare Mode​

A specialized mode for comparing human annotations with model outputs:

  • Side-by-side visualization of ground truth and predictions
  • Error analysis and quality assessment
  • Performance metrics and confidence visualization
  • Targeted refinement of problematic cases

Define Mode​

Used for setting up region of interest or preparing images for specialized workflows:

  • Define regions for focused annotation
  • Set up image preprocessing parameters
  • Configure specialized visualization settings
  • Prepare images for specific analysis tasks

Labeling Tools​

The interface provides a comprehensive set of annotation tools to handle various computer vision tasks:

Geometry Tools​

Tools for creating precise spatial annotations:

  • Polygon Tool: Create multi-point shapes to outline objects with irregular boundaries
  • Circle Tool: Draw circular annotations with adjustable radius
  • Bounding Box Tool: Create rectangular regions around objects
  • Oriented Bounding Box: Draw rotated rectangles for better fit to angled objects
  • Point Tool: Place precise keypoints or landmarks

Classification Tools​

Tools for assigning categories and tags:

  • Image Classification: Apply labels to the entire image
  • Region Classification: Assign categories to specific annotated areas
  • Multi-class Support: Apply multiple labels simultaneously
  • Hierarchical Labels: Work with nested classification structures

Editing Tools​

Tools for refining and modifying annotations:

  • Select & Move: Reposition existing annotations
  • Reshape: Adjust annotation boundaries
  • Copy & Paste: Duplicate annotations within or across images
  • Delete: Remove unnecessary annotations

Advanced Features​

Specialized capabilities for efficient annotation:

  • Label Presets: Save and reuse common annotation configurations
  • Automatic Snapping: Precisely align annotations to image features
  • Keyboard Shortcuts: Speed up workflow with customizable shortcuts
  • Batch Operations: Apply changes to multiple annotations simultaneously

Multi-user Collaboration​

The interface supports real-time collaboration among team members:

User Awareness​

Stay informed of team activity:

  • Active Users Panel: See who is currently working in the dataset
  • Position Indicators: Identify which images team members are viewing or editing
  • Status Updates: Receive notifications when others make significant changes
  • Color-Coded Identity: Each user has a distinct color for their annotations and cursors

Collaborative Workflow​

Efficiently distribute work among team members:

  • Label Assignments: Allocate specific labeling tasks to team members
  • Progress Tracking: Monitor completion status across the team
  • Conflict Resolution: Built-in mechanisms to prevent overlapping edits
  • Real-time Synchronization: See changes from other users immediately

Communication Tools​

Interact with team members within the app:

  • Annotation Comments: Attach notes or questions to specific annotations
  • Change Notifications: Get alerts when others modify your work
  • Approval Workflow: Submit work for review and receive feedback
  • Version Control: Track changes and maintain consistent dataset states

Keyboard Shortcuts​

The interface offers numerous keyboard shortcuts to speed up your workflow:

Navigation Shortcuts​

  • ← →: Navigate to previous/next image
  • ↑ ↓: Move between labels
  • Home/End: Jump to first/last image in dataset

Tool Shortcuts​

  • P: Select Polygon tool
  • B: Select Bounding Box tool
  • C: Select Circle tool
  • O: Select Oriented Bounding Box tool
  • L: Select Point (Location) tool

Editing Shortcuts​

  • Ctrl+Z / Ctrl+Y: Undo/Redo
  • Ctrl+C / Ctrl+V: Copy/Paste selected annotations
  • Delete: Remove selected annotations
  • Escape: Cancel current operation

View Shortcuts​

  • +/-: Zoom in/out
  • 1: Fit image to view
  • 2: Zoom to 100% (actual size)
  • Space+Drag: Pan when zoomed in
  • Tab: Toggle between toolboxes

Integration with Models​

The interface seamlessly integrates with trained models for visualization and comparison:

Prediction Visualization​

View model outputs directly on images:

  • Classification Results: See predicted categories and confidence scores
  • Detection Visualizations: View bounding boxes with confidence levels
  • Segmentation Maps: Examine pixel-level predictions with color overlays
  • Side-by-side Comparison: Compare ground truth annotations with model predictions

Inference Controls​

Adjust how predictions are displayed:

  • Confidence Thresholds: Filter predictions based on confidence level
  • Class Filtering: Show/hide predictions for specific classes
  • Visualization Options: Customize how predictions appear visually
  • Batch Processing: Run inference on multiple images

Refinement Workflow​

Use model predictions to enhance annotation efficiency:

  • Pre-annotation: Start with model predictions and refine them
  • Error Focus: Quickly identify and fix model mistakes
  • Quality Metrics: Track agreement between human annotations and model predictions
  • Model Improvement: Generate new training data focused on model weaknesses

Image Manipulation​

The interface provides tools for adjusting image display:

Zoom Controls​

Precisely control image magnification:

  • Zoom Slider: Continuously adjust zoom level
  • Preset Zoom Levels: Quickly access common magnifications
  • Fit to Window: Automatically size image to fit the display
  • Pixel-Perfect View: Examine images at actual resolution

Display Adjustments​

Optimize image visibility:

  • Brightness/Contrast: Adjust image appearance for better visibility
  • Color Channels: Toggle between RGB, grayscale, or individual channels
  • Region of Interest: Focus on specific parts of large images
  • Grid Overlay: Enable measurement grid for precise sizing

Metadata Display​

View detailed image information:

  • Cursor Position: See pixel coordinates under cursor
  • Pixel Values: View color or intensity values at cursor position
  • Image Properties: Display dimensions, format, and other metadata
  • Annotation Statistics: Count and measure annotations on the image

Tips & Best Practices​

Maximize your efficiency when working with the labelling interface:

Performance Optimization​

  • Close unnecessary toolboxes to free up screen space
  • Use keyboard shortcuts for common actions
  • Limit the number of images loaded in the carousel for faster navigation
  • Consider splitting very large datasets into smaller chunks

Annotation Efficiency​

  • Learn and use keyboard shortcuts for faster tool switching
  • Use appropriate zoom levels for different annotation tasks
  • Take advantage of copy-paste for repetitive annotations
  • Save custom tool presets for recurring annotation patterns

Collaborative Work​

  • Communicate with team members before starting shared labeling sessions
  • Check which images others are working on to avoid conflicts
  • Use consistent labeling conventions across the team
  • Regularly save your work to prevent data loss

Quality Control​

  • Regularly review your annotations at different zoom levels
  • Check for missing or incomplete annotations before moving to the next image
  • Use the comparison mode to validate your work against model predictions
  • Take advantage of batch operations for consistent corrections

Troubleshooting​

Solutions for common issues when working with the labelling interface:

Display Problems​

  • Images not loading: Check your internet connection and try refreshing
  • Slow performance: Close other applications and browser tabs
  • Rendering issues: Try a different browser or update your current one
  • Distorted visualization: Reset zoom and pan to default positions

Tool Issues​

  • Tools not responding: Make sure you're in label mode, not view mode
  • Can't select annotations: Check if the correct layer is active
  • Can't create new labels: Verify you have appropriate permissions
  • Annotations not saving: Check your connection and try saving manually

Collaboration Challenges​

  • Can't see other users: Refresh the page to update user status
  • Conflicting edits: Communication is key; coordinate with your team
  • Lost changes: Use undo/redo to recover recent changes
  • Synchronization issues: Save your work and refresh the application
