Datasets

Create a Dataset​

Creating a new dataset is simple:

  1. Navigate to your project
  2. Click on the Datasets tab
  3. Click the Create Dataset button
  4. Fill in the required information:
    • Name: A unique, descriptive name for your dataset
    • Type: Select the appropriate dataset type for your project needs
    • Description (optional): Add details about the dataset's purpose or contents

Creating a new dataset

Dataset Management​

The datasets list view provides an overview of all datasets in your project, with powerful management tools:

Dataset Actions​

  • Edit: Modify dataset name, description, or other properties
  • Duplicate: Create an exact copy of a dataset with all its images
  • Delete: Remove a dataset (requires confirmation)
  • Move: Transfer a dataset to another project

Dataset management interface

Uploading and Downloading Images​

Upload Options​

When the a dataset is create click over it to open the specific dataset page.

To upload image to the dataset click on the Add Images option in the dataset options. You can upload images from your local computer or by dragging and dropping them into the drop zone. Images can be uploaded with a .json file with same name containig the annotations.

Supported File Formats​

The following image formats are officially supported in VisionCloud:

FormatExtensions
JPEG.jpg, .jpeg
PNG.png
TIFF.tif, .tiff

Note: While the system accepts files up to 50 MB, for optimal performance we recommend keeping individual image sizes below 20 MB when possible.

Download Options​

  • Download Dataset: Export the entire dataset as a ZIP file
  • Download Selected Images: Export only specific images

Upload and download options

Image Management​

Image Selection Tools​

  • Bulk Select: Choose multiple images at once
  • Filter Options: Find images based on label status, creation date, etc.
  • Sort Options: Order by name, date, size, or other criteria

Image Actions​

When selecting one or more images, you can:

  • Classify: Assign classification labels to images
  • Unlabel: Clear all annotations from selected images
  • Ignore/Include: Mark images to be excluded from or included in training
  • Delete: Remove images from the dataset
  • Move: Transfer images to another dataset
  • Download: Download selected images

Available actions for selected images

Advanced Filtering​

The platform offers powerful filtering capabilities to help you find specific images:

Filter Categories​

  • Label Status: Filter by labeled, unlabeled, or partially labeled images
  • Label Content: Filter by images containing specific labels
  • Date Range: Filter by upload date or last modification date
  • Image Properties: Filter by size or file format
  • Metadata: Filter by specific metadata values

Filter Combinations​

Bulk Operations​

Efficiently manage large collections of images with bulk operations:

Image Transfer​

Move selected images between datasets:

  1. Select the images you want to move
  2. Click the Move action button
  3. Choose the destination dataset
  4. Select transfer options (copy vs. move, transfer annotations)
  5. Confirm the operation

Batch Labeling​

Apply labels to multiple images at once:

  1. Select the target images
  2. Click the Classify action button
  3. Choose the labels to apply
  4. Optionally adjust confidence levels
  5. Apply the labels to all selected images

The image would show the batch classification interface with multiple images and label selection

Bulk Processing​

Other bulk operations include:

  • Delete multiple images: Remove several images in one operation
  • Export selection: Download only the selected images
  • Ignore/Include multiple images: Mark multiple images as ignored or included

Label Configuration​

In the dataset options you can access the label configuration options for the dataset.

Label Management​

  • Create Labels: Define new annotation categories
  • Edit Labels: Modify existing label properties
  • Delete Labels: Remove unused or incorrect labels
  • Import Labels: Import lables from other datasets

Label Types​

VisionCloud supports multiple label types for different annotation needs:

  • Bounding Box: Rectangular regions around objects
  • Region: Multi-point shapes for irregular objects
  • Classification: Simple category assignment for whole images
  • Location: Precise object locations with sub-pixel accuracy
